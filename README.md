# GeoQuery - 地名经纬度批量查询与逆地理编码工具

批量查询地名经纬度（正向地理编码），也支持经纬度批量反查地址（逆地理编码）。上传 Excel/CSV 文件，自动调用高德地图 API 获取坐标并导出结果。

## 功能特点

- **正向/逆地理编码**：地名查坐标，或坐标查地址，一键切换
- **批量查询**：一次上传上百条数据，自动逐条查询
- **多 API Key 自动轮换**：可配置多个 Key，超出日配额自动切换下一个，查询不中断
- **配置持久化**：API Key 和模式设置自动保存，下次启动无需重新配置
- **文件格式**：支持 `.xlsx`、`.xls`、`.csv` 三种格式
- **实时进度**：查询过程中实时显示当前进度和每条结果
- **结果导出**：查询完成后可直接下载结果文件
- **开箱即用**：提供 Windows 独立可执行文件，无需安装 Python 环境

## 快速开始

### 方式一：直接运行（推荐）

从 [Releases](https://github.com/ydyydsediz/geoquery/releases) 页面下载最新版 `GeoQuery-windows.zip`，解压后运行 `GeoQuery.exe`，浏览器会自动打开操作页面。

### 方式二：从源码运行

```bash
# 克隆项目
git clone https://github.com/ydyydsediz/geoquery.git
cd geoquery

# 安装依赖
pip install -r requirements.txt

# 运行
python app.py
```

浏览器访问 `http://127.0.0.1:5000` 即可使用。

## 使用说明

### 第一步：获取高德 API Key

1. 打开 [高德开放平台](https://lbs.amap.com/)，点击右上角**注册/登录**（如已有账号直接登录）
2. 登录后进入**控制台**，点击左侧菜单 **「应用管理」→「我的应用」**
3. 点击**「创建新应用」**，填写应用名称（如 `GeoQuery`）和应用类型，点击「创建」
4. 在新建的应用卡片中，点击**「添加」**按钮添加 Key
5. 在弹窗中：
   - **服务平台**选择 **「Web 服务」**（重要！不是 Web 端(JS API)）
   - 填写一个 Key 名称
   - 点击「提交」
6. 创建成功后即可看到生成的 **Key 字符串**，复制备用

> **多 Key 配置**：如需更高日配额，可注册多个高德账号分别创建 Web 服务 Key，在页面中一行一个填入，程序会自动轮换使用。
>
> **配额说明**：高德 Web 服务 API 每个 Key 每日免费额度为 5000 次，超出后自动切换下一个 Key，全部用尽后会暂停等待次日重置。

### 第二步：准备数据文件

选择查询模式后，准备对应格式的数据文件（`.xlsx`、`.xls` 或 `.csv`）：

**正向地理编码（地名→坐标）**：第一列填写地名，无需表头：

| A列 |
|-----|
| 北京市天安门广场 |
| 上海市东方明珠 |
| 杭州市西湖 |

**逆地理编码（坐标→地址）**：第一列经度，第二列纬度，无需表头：

| A列 | B列 |
|-----|-----|
| 116.397428 | 39.90923 |
| 121.499718 | 31.239703 |
| 120.15507  | 30.274085 |

### 第三步：查询

1. 运行 `GeoQuery.exe`（或 `python app.py`），浏览器会自动打开操作页面
2. 在页面顶部选择查询模式：**「地名→坐标」**（正向）或 **「坐标→地址」**（逆地理）
3. 将 API Key 粘贴到输入框中（多个 Key 请一行一个）
4. 拖拽或点击上传第二步准备的数据文件
5. 点击**「开始查询」**，页面会实时显示查询进度

> **提示**：API Key 和模式选择会自动保存，下次启动程序时自动恢复。

### 第四步：下载结果

查询完成后，页面会显示成功/失败的统计信息，点击**「下载结果」**按钮即可保存结果文件。正向编码结果包含：地名、经度、纬度；逆编码结果包含：经度、纬度、地址。

## 项目结构

```
geoquery/
├── .github/
│   └── workflows/
│       └── release.yml   # GitHub Actions 自动打包发布
├── templates/
│   └── index.html        # 前端页面
├── .gitignore
├── app.py                # 后端主程序（Flask）
```

## 打包为可执行文件

```bash
# 一键打包（Windows）
build.bat
```

打包产物位于 `dist/GeoQuery/` 目录，运行 `GeoQuery.exe` 即可启动。
